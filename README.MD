# Jobster API

#### Starter

The starter is a copy of jobs-api final project, just with additional data.


- install dependencies

```sh
npm install
```

- create .env and provide correct values


.env

```js
MONGO_URI=
JWT_SECRET=
JWT_LIFETIME=
```

- start the project

```sh
npm start
```

- you should see "Server is listening ...." text
